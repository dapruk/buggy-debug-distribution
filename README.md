# BITEUK POS & Werhos - Debugging Competition

Welcome to the BITEUK POS & Werhos debugging competition! You have been given a production-ready application that contains several intentional bugs. Your goal is to find and fix them.

## Getting Started

### Prerequisites
- Node.js (v20 or later recommended)
- npm or pnpm
- Supabase CLI (optional, but helpful)

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.

## The Challenge

There are **22 bugs** hidden in this application. They range from simple UI glitches to complex backend logic errors.

- **Frontend**: React, React Query, Tailwind CSS
- **Backend**: Supabase (PostgreSQL + RPC)

Good luck!
